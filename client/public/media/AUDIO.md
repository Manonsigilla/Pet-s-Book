# Fichiers audio — cris d'animaux

Ces fichiers sont chargés à la demande (pas de préchargement) par le bouton
« Écouter un cri d'animal » sur la page d'accueil.

## Formats

Utiliser des **MP3 mono, 64 kbps, 1 à 3 secondes** pour rester sous 30 Ko par fichier.
Le format MP3 est privilégié (compatible tous navigateurs, plus léger que WAV/OGG).

## Liste des fichiers nécessaires

| Fichier | Son |
|---------|-----|
| `chat.mp3` | Miaulement de chat |
| `chien.mp3` | Aboiement de chien |
| `oiseau.mp3` | Chant d'oiseau |
| `coq.mp3` | Chant du coq |
| `vache.mp3` | Meuglement de vache |
| `mouton.mp3` | Bêlement de mouton |

## Où trouver des sons libres et gratuits

- https://freesound.org — filtres : licence Creative Commons 0, durée < 3s
- https://pixabay.com/sound-effects — gratuits, pas d'attribution requise
